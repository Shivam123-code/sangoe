# Sangoe Frontend Plan

## Project Goal

Build a **premium, conversion-focused, frontend-only** website for **Sangoe Business Growth Operating System™**.

Sangoe is positioned as a **Business Growth Operating System** for startups, MSMEs, and growing enterprises. The website should communicate one clear promise:

> **Build a Business That Runs Without You.**

This site is not a backend app. It is a polished, high-trust, frontend experience designed to:
- explain the product clearly,
- feel premium and investor-ready,
- drive demo bookings and trial signups,
- showcase product depth through elegant UX,
- and build credibility for Indian businesses.

## Recommended UI/UX Direction

### Visual Style
- premium SaaS look
- calm dark-on-light or rich light theme
- deep navy, charcoal, white, cool gray accents
- subtle gold or teal highlights for premium feel
- glassy cards only if used sparingly
- strong spacing and hierarchy

### Typography
- modern sans-serif for body
- refined display font for hero headings if desired
- bold numerical styling for metrics and scores

### Layout Principles
- large hero sections
- strong spacing
- modular card system
- visual rhythm with alternating section backgrounds
- sticky sub-nav on long pages
- clean comparison tables
- strong CTA repetition

### Motion Principles
Motion should feel:
- smooth
- intelligent
- professional
- not flashy
- not playful

Use:
- fade up
- slight scale on hover
- parallax on hero assets
- counters and metrics animation
- progress indicators
- section reveal on scroll
- smooth route transitions
- floating UI accents

Avoid:
- excessive bouncy motion
- overused spin effects
- distracting micro-animations
- loud neon effects
- cluttered motion everywhere
